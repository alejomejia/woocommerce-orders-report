This is a project scaffolded with the latest LTS version of Node using ES6 imports and TypeScript.

```
└── README.md
```

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/my-node-project.git`
2. Navigate into the project directory: `cd my-node-project`
3. Install the dependencies: `npm install`
4. Start the development server: `npm run dev`

## Scripts

- `dev`: Starts the development server with `nodemon`.
- `build`: Compiles the TypeScript code to JavaScript.
- `start`: Starts the server with the compiled JavaScript code.
